

# Landing Page Simples

Este projeto é uma landing page simples desenvolvida para praticar e consolidar conhecimentos em HTML e CSS. A página foi criada com foco na estruturação básica de elementos e na estilização utilizando CSS.

## Funcionalidades

- **Estrutura Simples:** A página é composta por uma estrutura básica em HTML, incluindo cabeçalho, seções de conteúdo e rodapé.
- **Estilização com CSS:** O design da página foi feito utilizando CSS para organizar o layout e definir cores, fontes e espaçamentos.

## Tecnologias Utilizadas

- **HTML5:** Para a estruturação do conteúdo da página.
- **CSS3:** Para a estilização e layout da página.

## Como Usar

1. Clone o repositório para a sua máquina local:

   ```bash
   git clone https://github.com/seu-usuario/landing-page-simples.git
   ```

2. Abra o arquivo `index.html` no seu navegador preferido.

## Estrutura do Projeto

```bash
├── index.html        # Estrutura HTML da página
└── style.css         # Estilos CSS da página
```
